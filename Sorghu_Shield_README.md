# Sorghu-Shield: Innovative & Cost-Effective Eco-Friendly Food Packaging

## 🌱 Overview
Sorghu-Shield is an eco-friendly food packaging solution developed using sweet sorghum bagasse. It aims to replace plastic containers with biodegradable, cost-effective, and durable alternatives suitable for real-world food applications.

## 🚨 Problem Statement
- Plastic packaging causes severe environmental pollution  
- Existing paper/bagasse containers fail with hot food  
- Lack of strong biodegradable alternatives  

## 💡 Proposed Solution
- Use sweet sorghum bagasse fiber  
- Manufacture leak-proof containers  
- Replace plastic packaging  

## 🚀 Innovation
- High-temperature hot pressing  
- Leak-proof design  
- Fully biodegradable  

## ⚙️ Manufacturing Process
Raw Material → Cleaning → Crushing → Pulping → Molding → Vacuum → Hot Press → Trimming → Sterilization → Packing  

## 📊 Financials
- Cost per box: ₹1.90  
- Selling price: ₹3.00  
- Profit: ₹1.10  

## 🏁 Conclusion
Sorghu-Shield is a sustainable and cost-effective alternative to plastic packaging.
